# ORION

ORION is a modular C++ console application built to practice programming, software structure, and system design concepts.

## Features

### Student Management System
- Add students
- View all students
- Search students
- Edit student details
- Delete students

### Banking System
- Create accounts
- Deposit money
- Withdraw money
- Check balance

### Utilities
- Calculator
- Extra utility tools

### Games
- Console-based mini games

### Admin Panel
- Administrative controls and management

---
